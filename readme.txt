Text Analysis Script README

Introduction
This script performs textual analysis on data obtained from URLs. It extracts text content from web pages, cleans the data,
and analyzes various metrics such as sentiment, readability, word count, syllable count, personal pronoun count, and average word length.
The results are stored in an Excel file for further analysis.

Prerequisites
Python 3.x
Required Python libraries: nltk, requests, beautifulsoup4, pandas

Run the script:
-> Open a terminal or command prompt.
-> Navigate to the directory where the script is located.
-> Run the script using the following command:
--> Copy code
--> python text_analysis_script.py

Output:
-> After execution, the script will generate an Excel file named output_data.xlsx in the output_data_structure directory.
-> This file contains the analyzed data
